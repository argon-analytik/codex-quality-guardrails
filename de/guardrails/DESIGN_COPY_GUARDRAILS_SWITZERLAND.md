# COPY GUARDRAILS

## Sichtbarer deutscher Text
Für allen deutschsprachigen, sichtbaren Text in UI, E-Mails, PDFs, Portalen, Notifications, Assistenten-Antworten und sonstiger Kommunikation gilt:
- ä ö ü statt ae oe ue
- ss statt ß
- Schweizer Typografie
- « » statt " " in sichtbarer Kommunikation, wo typografisch passend
- ‹ › für einfache Anführungen, wo typografisch passend

## Ausnahmen
ASCII-nahe Schreibweisen wie ae, oe, ue sind erlaubt in:
- Code
- Identifiers
- Slugs
- URLs
- Dateinamen
- Pfaden
- CLI-Befehlen
- technischen Schlüsseln
- kebab-case / snake_case

## Regel
Technische Strings und menschliche Sprache klar unterscheiden.
Menschlich sichtbare Sprache soll wie saubere, korrekte Schweizer Typografie wirken.
